:שלושה הבדלים בין הגירסאות השונות


  • הבדל במטרות-
    WEB 1.0 : נוצרה במטרה לשתף מידע ולחבר כמה שיותר אנשים למידע
    WEB 2.0: נוצרה במטרה לחבר בין אנשים
    WEB 3.0: נוצרה במטרה לחבר ידע, כלומר להנגיש את המידע לאנשים
  • הבדל במנועי החיפוש-
    WEB 1.0: מנועי החיפוש חיפשו מאקרו תוכן, לכן החיפוש היה מהיר מאוד אבל הרבה פעמים הוחזרו מסמכים לא רלוונטיים. כלומר, הדיוק בתוצאות החיפוש היה נמוך.
    WEB 2.0: מנועי החיפוש מחפשים מיקרו תוכן ותגיות. תהליך התיוג מתבצע ידנית, והוא ארוך ומייגע, ולכן מכסה רק חלק קטן מהאינטרנט. החיפוש לא מהיר כמו בגירסה הראשונה, אך מדוייק יותר.
    WEB 3.0: מנועי החיפוש יחפשו מיקרו תוכן ותגיות, אשר יתוייגו באופן אוטומטית. תוכן המאקרו של הגירסה הראשונה יומר לתגיות מיקרו, בכך הדיוק של תוצאות החיפוש יעלה.
  • הבדל ביצירת התוכן-
    WEB 1.0: יוצרי התוכן (האתרים) היו בעיקר חברות שפירסמו תוכן כדי שמשתמשים יצרכו אותם
    WEB 2.0: החברות יצרו פלטפורמות שבהן אנשים פרסמו תוכן לאנשים אחרים. בנוסף, אנשים פרסמו תוכן שאנשים אחרים יכולים לצרוך.
    WEB 3.0: אנשים יוצרים אפליקציות שבהם אנשים נוספים/אחרים יכולים לתקשר. כמו כן, חברות יוצרות פלטפורמות בהן אנשים יכולים לפרסם שירותים ע"י ניצול הקשרים בין אנשים או תוכן חברתי.
